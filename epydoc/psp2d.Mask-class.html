Builds a new Mask from an Image part.

Parameters:
  • img (Image) - Source image.
  • x (integer) - X position to start from
  • y (integer) - Y position to start from
  • w (integer) - Width of the rectangle to consider.
  • h (integer) - Height of the rectangle to consider.
  • t (integer, from 0 to 255) - Gray threshold. For each pixel whose gray value is under this value, the corresponding mask bit will be 0, else 1.
