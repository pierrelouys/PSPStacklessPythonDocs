Function Details

connectToAPCTL(config=1, callback=None, timeout=-1)

 

Connects to the network.

Parameters:
  • config (integer) - The network configuration to use, starting at 1. See enumConfigs.
  • callback (callable) - An optional callback to be invoked at each state change in the connection process. It must take an integer state as argument.
  • timeout (integer) - Connection timeout

getAPCTLState()

 

Returns the current connection state. This is the same value as the one passed to the callback in connectToAPCTL.

enumConfigs()

 

Lists all available network configurations.

Returns:
