Creates a new Transform object. There are actually two constructors:

  • Transform(type) creates a new standard transformer
  • Transform(callable, param) creates a new user-defined transformer

For standard transformers, type may be one of:

  • TR_PLUS: Adds param to each color component in the image
  • TR_MULT: Multiplies each color component by param
  • TR_G2A: For each pixel, set the alpha level to the gray level
  • TR_GRAY: Grays the image
  • TR_BW: Converts the image to monochrome, using param as a threshold.

User transforms must provide a callable. It will be called for each pixel to transform, and given as arguments:

  • The pixel X position
  • The pixel Y position
  • The pixel value (a Color instance)

It must return a new Color value for the pixel.
