Method Details

blit(self, src, sx=0, sy=0, w=-1, h=-1, dx=0, dy=0, blend=False, dw=-1, dh=-1)

 

Blits an image to this one. You can blit either:

  • An image to an image
  • An image to the screen
Parameters:
  • src (Image) - Source image
  • sx (integer) - Source X position
  • sy (integer) - Source Y position
  • w (integer) - Source width (if -1, as much as possible)
  • h (integer) - Source height (if -1, as much as possible)
  • dx (integer) - Destination X position
  • dy (integer) - Destination Y position
  • blend (boolean) - If True, blending will be performed based on the source's alpha channel.
  • dw (integer) - Destination width; if -1, don't stretch
  • dh (integer) - Destination height; if -1, don't stretch

Creates a new image. There are actually three ways to call this constructor:

  • Image(filename): Loads from a PNG file
