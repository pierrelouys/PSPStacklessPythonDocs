Class Hierarchy

  • object: The most base type
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • pspsnd.Error: Generic error for this module
        • psp2d.Error: Generic exception for this module
    • psp2d.Color: An RGBA color
    • psp2d.Controller: This class is used to read the PSP controller state (buttons, stick).
    • psp2d.Filter: This class isn't actually available in the psp2d module, but is a generic abstract parent for objects that take an Image as input and return another Image.
      • psp2d.DifferenceFilter: Image difference filter
      • psp2d.DilationFilter: Morphologic dilation filter.
      • psp2d.ErosionFilter: Morphologic erosion filter.
      • psp2d.GrayFilter: Converts images to gray level.
      • psp2d.MonochromeFilter: Converts images to monochrome.
      • psp2d.ScaleFilter: Image scaling filter, with linear interpolation.
      • psp2d.SobelFilter: Sobel gradient filter.
      • psp2d.SubFilter: Image cropping filter.
    • psp2d.Font: An SFont-like font.
    • psp2d.Image: An image class.
    • psp2d.InPlaceFilter: This class isn't actually available in the psp2d module, but is a generic abstract parent for objects that modify an Image in-place.
      • psp2d.GrayInPlaceFilter: In-place version of GrayFilter.
      • psp2d.MonochromeInPlaceFilter: In-place version of MonochromeFilter.
    • psp2d.Mask: This is actually a two-dimensional boolean array, used for collision detection.
    • pspsnd.Music: Music playback
    • pspsnd.Sound: Sound effects.
    • psp2d.StructuringElement: This is a structuring element for morphologic operations; see ErosionFilter and DilationFilter.
    • psp2d.Timer: TODO
    • psp2d.Transform: Simple image transformation class.
